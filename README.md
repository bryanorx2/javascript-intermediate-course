# JavaScript Intermediate Course 🚀

Practical exercises from the intermediate JavaScript course, following [MoureDev](https://github.com/mouredev)'s learning path.

## 👨‍💻 Author

**Bryan** — [@bryanorx2](https://github.com/bryanorx2)

## 📚 Course Content

1. **Advanced Functions** — First-class functions, Arrow functions, IIFE, Rest parameters, Spread, Closures, Recursion, Partial functions, Currying, Callbacks
2. **Advanced Data Structures** — Advanced Arrays, Sets and Maps (functional methods, manipulation, sorting, searching, iteration, conversion)
3. **Advanced Objects & Classes** — Prototypes, Inheritance, Static methods, Polymorphism, Mixins, Singleton, Symbol, Proxy
4. **Asynchronous JavaScript** — Event Loop, Callbacks, Promises, Async/Await
5. **APIs** — REST API, HTTP Methods, Fetch, Error handling, Authentication
6. **DOM** — Selection, Manipulation, Events, Practical examples
7. **Debugging** — Debugging tools
8. **Regex** — Syntax and methods: test, replace, exec
9. **Testing** — Jest
